IsBackPlateSource
Gets a value indicating whether this instance is back plate source.

Only one cache handle inside one cache manager can be back plate source. Usually this is a distributed cache. It might not make any sense to define an in process cache as back plate source.

If no back plate is configured for the cache, this setting will have no effect.
