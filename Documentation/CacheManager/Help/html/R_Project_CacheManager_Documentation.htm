CacheManager is a common interface and abstraction layer for caching. It supports various cache providers and implements many advanced features.
Namespaces
NamespaceDescription
CacheManager.AppFabricCache
This namespace contains the Microsoft AppFabric cache handle.
CacheManager.Core
The base namespace for the CacheManager.Core package contains all interface definitions for the CacheManager.
CacheManager.Core.Cache
This namespace contains the base implementations for CacheManager and CacheHandle. It is the base implementation of the CacheManager library.
CacheManager.Core.Configuration
This namespace contains all configuration related implementations for the CacheManager.
CacheManager.Couchbase
This namespace contains the Couchbase cache handle base on the Couchbase .Net client.
CacheManager.Memcached
This namespace contains the Memcache cache handle base on Enyim's memcached client.
CacheManager.Redis
This namespace contains the basic Redis helpers and configuration classes. It implements a Redis configuration section which can be used to confgure the Redis cache handle via app/web.config.
CacheManager.Serialization
 
CacheManager.SystemRuntimeCaching
This namespace contains the MemoryCache cache handle build on top of System.Runtime.Caching in memory cache implementation.
CacheManager.Web
This namespace contains System.Web related implementations based on CacheManager.
CacheManager.WindowsAzureCaching
This namespace contains the Microsoft Azure cache handle. It shares the implementation with AppFabricCache but uses different Nuget references.
