CacheManager.Redis Namespace
This namespace contains the basic Redis helpers and configuration classes. It implements a Redis configuration section which can be used to confgure the Redis cache handle via app/web.config.
Classes
  ClassDescription
Endpoint
Configuration section for the CacheManager.
EndpointCollection
Collection of end point configurations.
RedisCacheBackPlate
Implementation of the cache back plate with Redis Pub/Sub feature.

Pub/Sub is used to send messages to the redis server on any Update, Cache Clear, Region Clear or Remove operation. Every cache manager with the same configuration subscribes to the same chanel and can react on those messages to keep other cache handles in sync with the 'master'.

RedisConfiguration
RedisConfiguration will be used for configuring e.g. StackExchange.Redis by code or configuration file.

The element was added only because StackExchange.Redis doesn't support configuration via web/app.config

RedisConfigurationBuilder
RedisConfigurationBuilder helps creating the RedisConfiguration object via code.
RedisConfigurations
Manages redis client configurations for the cache handle.

Configurations will be added by the cache configuration builder/factory or the configuration loader. The cache handle will pick up the configuration matching the handle's name.
